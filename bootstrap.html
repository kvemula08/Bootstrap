Welcome to Bootstrap Tutorial

1. What is BootStrap?

Twitter Bootstrap is the most popular front end framework in the recent time. It is sleek, intuitive, and powerful mobile first front-end framework for faster and easier web development. It uses HTML, CSS and Javascript.

2. Why Use Bootstrap?

a. Mobile first approach : Bootstrap 3, framework consists of Mobile first styles throughout the entire library instead them of in separate files.

b. Browser Support : It is supported by all popular browsers.

c. Provides a clean and uniform solution for building an interface for developers.

c. It contains beautiful and functional built-in components which are easy to customize.
